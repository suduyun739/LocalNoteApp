# 🚀 快速开始指南

## 📥 第一步：打开应用

### 方法 1：直接打开（最简单）
1. 找到 `index.html` 文件
2. 双击打开（会用默认浏览器打开）
3. 开始使用！

### 方法 2：拖放到浏览器
1. 打开你的浏览器（Chrome/Firefox/Edge）
2. 将 `index.html` 文件拖放到浏览器窗口
3. 开始使用！

### 方法 3：使用本地服务器（推荐）
```bash
# 进入项目文件夹
cd LocalNoteApp

# 使用 Python 3（如果已安装）
python -m http.server 8000

# 或使用 Node.js（如果已安装）
npx http-server -p 8000

# 然后在浏览器访问
http://localhost:8000
```

## 🎯 5分钟上手教程

### 1️⃣ 导入示例数据（可选）

如果想快速体验，可以导入我们准备的示例数据：

1. 点击右上角的 **"📤 导入"** 按钮
2. 选择 `sample-data.json` 文件
3. 点击确认
4. 现在你有 8 条示例笔记了！

### 2️⃣ 浏览笔记

- 在左侧点击不同的类型（📚 读书、🎬 观影、📅 日常）来筛选
- 点击任意笔记卡片查看详情
- 注意右侧的笔记数量统计

### 3️⃣ 搜索笔记

1. 在顶部搜索框输入关键词，如 "科幻"
2. 实时看到匹配的笔记
3. 点击 ✖ 清除搜索

### 4️⃣ 创建新笔记

**创建读书笔记：**
1. 点击左下角 **"➕ 新建笔记"**
2. 选择 **"📚 读书笔记"**
3. 填写表单：
   - 标题：《活着》读后感
   - 书名：活着 ⭐
   - 作者：余华
   - 评分：5
   - 日期：选择今天
   - 内容：写下你的感想...
   - 标签：文学,经典,中国
4. 点击 **"💾 保存笔记"**

**创建观影笔记：**
1. 点击 **"➕ 新建笔记"**
2. 选择 **"🎬 观影笔记"**
3. 填写：
   - 标题：《阿甘正传》观后感
   - 影片名称：阿甘正传 ⭐
   - 导演：罗伯特·泽米吉斯
   - 主要演员：汤姆·汉克斯
   - 评分：5
   - 内容：生活就像一盒巧克力...
   - 标签：励志,经典
4. 保存

**创建日常笔记：**
1. 点击 **"➕ 新建笔记"**
2. 选择 **"📅 日常笔记"**
3. 填写标题、日期和内容即可
4. 保存

### 5️⃣ 编辑和删除

**编辑笔记：**
1. 点击笔记卡片进入详情页
2. 点击右上角 **"✏️ 编辑"**
3. 修改内容
4. 保存

**删除笔记：**
1. 在详情页点击 **"🗑️ 删除"**
2. 确认删除
3. 笔记被删除，返回列表

### 6️⃣ 排序笔记

点击右上角的排序下拉菜单：
- **最新优先**：按日期从新到旧
- **最旧优先**：按日期从旧到新
- **标题 A-Z**：按标题字母顺序

## 💾 数据备份

### 导出备份

定期导出数据是个好习惯！

1. 点击右上角 **"📥 导出"**
2. 自动下载 JSON 文件
3. 文件名格式：`notes-backup-2024-01-19.json`
4. 保存到安全位置（建议保存到云盘）

### 恢复数据

如果需要恢复或迁移数据：

1. 点击右上角 **"📤 导入"**
2. 选择之前导出的 JSON 文件
3. **注意**：这会替换当前所有数据！
4. 确认导入

## 🔧 问题排查

### 笔记不显示？

1. 检查浏览器控制台（F12）是否有错误
2. 确认 IndexedDB 已启用（现代浏览器默认开启）
3. 尝试刷新页面（F5）

### 数据丢失？

1. 检查是否误清除了浏览器数据
2. 查看是否有备份的 JSON 文件
3. 使用导入功能恢复数据

### 导入失败？

1. 确认 JSON 文件格式正确
2. 文件应该是数组格式：`[{...}, {...}]`
3. 检查文件大小（不要超过浏览器限制）

### 样式错乱？

1. 确保 `styles.css` 文件在同一文件夹
2. 检查浏览器是否支持 CSS Grid 和 Flexbox
3. 尝试使用最新版本的 Chrome/Firefox/Edge

## 💡 使用技巧

### 标签的使用

标签能帮助你更好地组织笔记：

- **按主题**：科幻、文学、历史
- **按重要性**：必读、精选、推荐
- **按状态**：已读、在读、想读
- **按心情**：感动、震撼、思考

### 搜索技巧

- 搜索多个关键词：输入"科幻 哲学"会搜索包含这些词的笔记
- 搜索书名或影名：直接输入作品名称
- 搜索标签：输入标签名可以找到相关笔记

### 笔记组织

- **读书笔记**：记录读后感、金句摘抄、思考
- **观影笔记**：记录观后感、经典台词、情节分析
- **日常笔记**：记录生活、学习、工作心得
